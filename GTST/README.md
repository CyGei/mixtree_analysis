This folder is not part of the manuscript. It explores the [GTST python package](https://openresearchsoftware.metajnl.com/articles/10.5334/jors.478) which also offers tools to compare collections of graphs.

In this folder you can find:

\- `mixtree_pipeline.R`: an example script that uses `mixtree` to compare collections of graphs. It saves a few forests to be used as input for `GTST`. And later compares the output of `mixtree` with `GTST`.

\- `gtst_pipeline.py`: an example script that uses `GTST` to compare collections of graphs.
